## Project Overview
This project focuses on exploratory data analysis (EDA) of a vehicle dataset
to understand factors affecting vehicle pricing.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Analysis
- Data cleaning and preprocessing
- Handling missing and incorrect values
- Price analysis based on engine size and horsepower
- Mileage vs price analysis
- Visualization of correlations and trends

## Outcome
Derived meaningful insights to understand the impact of vehicle features
on pricing and performance.
